## Overview

The `python-console-snake` project is a lightweight, configurable version of the Snake game designed to run in the terminal/console using Python. The game is easy to use, modify, and extend, making it ideal for learning and for entertainment purposes during the first semester.

## Features

- Runs directly in any console using Python.
- Configurable game size and themes.
- Multiple themes included (classic, minimal, jungle) and easy customization to add more.
- Open-source (MIT License) for learning and modification.

## Usage

1. Clone the repository:  
   `git clone https://github.com/helpershyam/python-console-snake.git`
2. Change into the project directory:  
   `cd python-console-snake`
3. Run the game:  
   `python snake`

For additional options, run:  
`python snake -help`
